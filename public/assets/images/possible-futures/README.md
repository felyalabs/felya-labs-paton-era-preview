# Possible Futures runtime assets

Only the optimized WebP sketches required by the live website are kept in `public/` and copied into the production build.

Original PNG sketches, legacy full-scene blueprints, and generated mask experiments live in `assets-source/possible-futures/` and are not deployed.

Sketch files use the pattern `<motif>-<theme>.<format>`, where `theme` is `black` for light mode and `white` for dark mode.
